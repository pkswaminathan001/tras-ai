# 🎯 TRAS AI — Business Analyst Resume Analyzer

**AI-powered resume-to-job matching for Business Analyst and Insurance Operations roles.**

## Business Problem
Job seekers spend hours manually comparing resumes against job descriptions. Recruiters need to quickly identify qualified candidates. TRAS AI solves both problems.

## Solution
| Feature | What it does |
|---------|--------------|
| ATS Score | Evaluates resume compatibility with automated screening |
| Match Score | Overall fit for the role (0-100) |
| Missing Keywords | Identifies gaps to improve resume |
| Resume Rewrites | Specific suggestions for each bullet point |
| LinkedIn Headline | Tailored for the target role |
| Interview Questions | Predicted questions based on the JD |
| Salary Range | Market estimate for Luxembourg/NL/IE |

## Architecture
- Frontend: GitHub Pages
- AI: Google Gemini API (user provides key)
- No backend — runs entirely in browser
