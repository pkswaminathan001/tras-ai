# Contributing to TRAS v1.0

Thanks for your interest! This is a personal portfolio project built to demonstrate AI+BFSI skills for Luxembourg roles.

## 🛠️ How to Run Locally
1. Clone the repo
2. Open `index.html` in your browser (no build step needed)
3. Or use VS Code Live Server extension

## 🤝 Want to Contribute?
1. Fork the repo
2. Create a feature branch (`git checkout -b feat/your-idea`)
3. Commit changes (`git commit -m 'feat: your improvement'`)
4. Push and open a Pull Request
